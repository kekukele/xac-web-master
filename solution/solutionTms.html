Terminal Management System 2.0(TMS 2.0)

XAC TMS II is XAC's next generation Terminal Management System, and is also part of XAC's Cloud DMS. Developed by Google Firebase SDK which make it is the ideal TMS system for managing Android Devices.

Intuitive Management UI for Multi Estate Management Capabilities

It empowers user with intuitive management UI and efficient operation tool for below estate management capabilities:

  • Enrollment
  • Device Management
  • Device Groups Management
  • Application Management
  • Package Management
  • System Action
  • Scheduled Action
  • Event Management
  • Profile Management
  • Auto Configuration

Real Time Update and Auto-Configuration

Real time update and Auto-configuration are 2 key features in XAC Cloud TMS to help our customer easier to manage your device estate at a lower cost.

  • Real Time Updates
    • OS
    • Kernel
    • APK
    • Application
    • Parameters
  • Auto-Configuration: For lower total cost of ownership and a better “Out of the Box” experience, TMS 2.0 Auto Configuration enables configuring any device’s software (OS, Firmware, Applications, and Profiles) remotely when the device first connects to the network.

The device can be shipped directly to merchants without pre-staging/configuring the device in the local warehouse before shipment. Easy to deploy devices without the need for installers!

Customer just needs to define Software Configuration (OS, Firmware, Applications, and Profiles) from XCE TMS2.0 Web Portal for any group of devices.

Operation Efficiency Increase and Lower Total Cost of Ownership

XAC TMSII is a powerful and comprehensive system that can be used at development phase, deployment phase and even Maintenance phase. By utilizing XAC’s TMS II, it can help our customer to increase overall operation efficiency and also lower the Total Cost of Ownership.
